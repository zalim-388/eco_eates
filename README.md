
# Eco Eates 🍽️  

A simple **food ordering app** built using **Flutter**. This app showcases a list of food items with cart functionality.  

## 🚀 Features  
✅ Product Listing / Food Menu  
✅ Add to Cart & Checkout System  
✅ Clean & Simple UI
